travGen

sector{}
The sector object contains all the information about a given sector including metadata, worlds data, and trade information.

Structure

metadata{}
Information about the sector
	Properties
	name      String  None   Name of the sector
	density   int     0-100  Percentage change a world exists
	maturity  int     0-3    Level of maturity

worlds[]
Array of world objects

secMap{}
Object that tracks which hexes have worlds

subsectors[]
Array of subsector objects

tradeRoutes[]
Array of route objects
